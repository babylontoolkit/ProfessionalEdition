Babylon Editor Toolkit - Visual Studio Code
==================================================

* Install the .NET SDK [https://dotnet.microsoft.com/download] which includes the Runtime and the dotnet command.

- [Windows only] Logout or restart Windows to allow changes to %PATH% to take effect.

- [macOS only] To avoid seeing "Some projects have trouble loading. Please review the output for more details", make sure to install the 

- latest stable Mono [https://www.mono-project.com/download/] release.

- Note: This version of Mono, which is installed into your system, will not interfere with the version of MonoDevelop that is installed by Unity.


* Install the C# extension [https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp] from the VS Code Marketplace.
